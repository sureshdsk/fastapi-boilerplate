# fastapi-starter-kit

```
git clone https://github.com/sureshdsk/fastapi-starter-kit.git

cd /path/to/project-root

# create virtualenv
virtualenv -p python3 venv

# activate virtualenv
. venv/bin/activate

# install dependancies
pip install -r requirements.txt

# run app
./run.sh 
./run_async.sh # runs async version
```
